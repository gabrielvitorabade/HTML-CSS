Principais formatações

Negrito/ Destaque

Nesta frase temos um termo em negrito usando a tag b.

Já aqui temos um termo em destaque usando a tag strong.

Itálico/ ênfase

Aqui temos duas palavras em itálico ultilizando a tag i.

Já neste outro exemplo temos esses termos em ênfase ultilizando a tag em.

Marcador de texto

Podemos também marcar um trecho ultilizando a tag mark, e alterar a cor do marcador ultilizando parametros de css

texto grande e texto pequeno

Podemos deixar um texto pequeno usando a tag small
E um texto grande usando a tag big porém essa tag esta obsoleta.

Texto deletado e Texto inserido

Para marcar um texto como deletado para que seja desconsiderado assim, deve-se usar a tag del.

E para indicar um texto inserido usa-se a tag ins. Ou a tag u, que não é indicada.

Texto subscrito e texto sobrescrito

Para sobrescrever um algarismo como o 3 em 103+10 é necessário usar a tag sup.
Já para subscrever como o 2 em H2O usa-se a tag sub.
